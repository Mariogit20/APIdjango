# APIdjango
Utilisation des 3 API django qui sont l'API Produit List ( http://127.0.0.1:8000/produits/ ) , l'API Contact List ( http://127.0.0.1:8000/contact/ ) , l'API Slider List ( http://127.0.0.1:8000/sliders/ ) . Par défaut l'Api Root de DJANGO RESTFRAMEWORK correspond à l'API Produit List ( "produits": "http://127.0.0.1:8000/produits/" )
